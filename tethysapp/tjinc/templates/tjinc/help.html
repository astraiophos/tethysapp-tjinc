    Purpose

    The purpose of this code is to create an interactive online topographic map that displays the watershed delineation from user-defined outlet points within Utah County. The map also displays previously determined flow-lines based of a DEM of the region. These flowlines are meant for the user to know where to click in order to delineate a watershed. Two additional pages are provided in the side menu for the benefit of the user as help files, including an end user document and technical specifications document.

    Script Acquisition

    The watershed delineation tool was formulated in ModelBuilder in ArcMap 10.3. It was then uploaded to the BYU geoserver and is available at geoserver.byu.edu under the TJ Dam Power folder. This tool was then uploaded to the tethys app from various scripts written in HTML and javascript, which were built in a virtual machine (Oracle VM VirtualBox). This code was written in PyCharm. The web app is designed to run in the Tethys portal. If run locally, Tethys must be launched in the virtual machine terminal prior to opening the web browser to view the webpage. Contact either Jacob Fullerton or Trey McGhin to obtain the code at jacobbfull@gmail.com and creden791@gmail.com respectively.

    Using the Web Application

    Once the web application is launched in your browser, the topographic base map will appear with the previously determined flowlines on top, as indicated in Figure 3. In order to delineate the watershed of the user’s choosing, the user will simply need to click with the cursor at any point on the flowlines. Please keep in mind that the user will need to zoom close to be sure to click on the flowlines. Otherwise, the user could click off of the flowlines and not receive an accurate result. Once the user has clicked on the flowlines, the tool will begin the geoprocessing. A progress status will indicate this to the user. In few minutes, the resulting delineated watershed will appear on the topographic map. The spatial extent of this watershed can then be compared to the region by the topographic base map (Figure 4).
